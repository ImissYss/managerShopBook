# Quản lý shop bán sách
**Tính năng ADMIN**

- Quản lý người dùng
- Quản lý sách(tác giả, thể loại,...)
- Quản lý comment, rating
- Quản lý đơn hàng

**Tính năng USER**

- Đăng nhập, đăng ký
- Mua sách
- Nhận xét, rating sách
- Tìm kiếm sách, tác giả

## Công nghệ sử dụng
**Front-end**

HTML + CSS + JS + BOOTSTRAP

**Back-end**

SPRING BOOT + SPRING DATA JPA + MYSQL + APACHE TILES + JSTL
